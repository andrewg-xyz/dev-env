# dev-env

Development environment configuration and utilities. A collection of gadgetry to make it easier to port my development environment.

## Danger Zone

This is poorly maintained :/

```sh
git clone <this repo>
cd dev-env
```

On OSX
```sh
./configure.sh
gsd configure
```

On Linux
```sh
./linux/init.sh
#log out and back in, for zsh to take effect
./linux/configure.sh
```
